# Task Flow Lite

A modern Angular learning project focused on clean architecture, Signals, and job-ready development practices.

## Tech Stack

- Angular (Standalone)
- TypeScript
- SCSS
- pnpm
- Husky + lint-staged + Prettier

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- pnpm

### Install & Run

```bash
pnpm install
pnpm start
```
